# Style: Warm (温馨风格)

## 视觉特征

- **色调**：温暖、友好、舒适
- **氛围**：亲切、温馨、有爱
- **适用**：生活、情感、家庭、温情类内容

## 色彩方案

### 主色调
- 暖橙：`#FFB74D`
- 珊瑚粉：`#FF8A80`
- 奶油黄：`#FFF59D`
- 浅棕：`#BCAAA4`

### 辅助色
- 米白：`#FFF8E1`
- 暖灰：`#EFEBE9`
- 浅橙：`#FFE0B2`

### 强调色
- 深橙：`#FF6F00`
- 玫瑰红：`#E91E63`

## 设计元素

### 图形
- 圆润形状
- 心形元素
- 柔和曲线
- 温暖图标（家、心、拥抱）

### 线条
- 中等粗细（2-3px）
- 圆润曲线
- 手绘感线条

### 装饰
- 爱心点缀
- 温暖图标
- 柔和阴影
- 渐变效果

## 排版风格

### 字体
- 圆润字体
- 手写风格
- 中等字重

### 布局
- 居中对齐
- 温馨氛围
- 柔和间距

## Prompt 模板

```
温馨友好风格，暖色调背景（暖橙、珊瑚粉、奶油黄），
圆润形状，柔和曲线，心形元素点缀，
手绘感线条，温暖图标（家、心、拥抱），
圆润字体，柔和阴影，渐变效果，
亲切舒适的视觉氛围
```

## 适用场景

- 生活故事
- 情感表达
- 家庭温情
- 亲子内容
- 温暖回忆
- 友情爱情

## 情绪关键词

温馨、友好、亲切、舒适、有爱、温暖、柔和、治愈
