# Layout: Balanced (平衡布局)

## 特点

- **信息密度**：中等（3-4个要点）
- **视觉平衡**：信息与留白均衡
- **适用场景**：文章配图、常规内容、通用场景

## 布局规则

### 信息量控制
- 3-4 个核心要点
- 适度留白
- 多个视觉元素平衡分布

### 视觉层次
- 主要元素占据 40-50% 画面
- 次要元素占据 20-30%
- 留白占据 20-30%

### 文字处理
- 标题：中等字号
- 正文：3-4段简短文字
- 关键词：3-5个

## 适用内容类型

- 文章配图（默认）
- 教程说明
- 产品介绍
- 常规内容
- 通用场景

## 示例结构

```
┌─────────────────────────┐
│   [标题]                 │
│                         │
│  [要点1]    [要点2]      │
│                         │
│  [要点3]    [要点4]      │
│                         │
│   [总结/CTA]             │
└─────────────────────────┘
```

## Prompt 提示词

在生成图片时，添加以下约束：

- "平衡布局，信息与留白均衡"
- "3-4个核心要点，清晰分布"
- "主要元素占据40-50%画面"
- "适度文字说明，易于阅读"
- "视觉舒适，不拥挤不空旷"
