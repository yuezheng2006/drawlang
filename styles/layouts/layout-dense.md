# Layout: Dense (密集布局)

## 特点

- **信息密度**：高（5-8个要点）
- **知识卡片**：适合知识整理、学习笔记
- **适用场景**：知识总结、概念图、学习资料

## 布局规则

### 信息量控制
- 5-8 个核心要点
- 少量留白
- 高信息密度

### 视觉层次
- 主要元素占据 60-70% 画面
- 次要元素占据 20-30%
- 留白占据 10-20%

### 文字处理
- 标题：中等字号
- 正文：5-8段简短文字
- 关键词：6-10个
- 使用编号、图标辅助

## 适用内容类型

- 知识卡片
- 学习笔记
- 概念总结
- 技术文档
- 信息整理
- 思维导图

## Prompt 提示词

在生成图片时，添加以下约束：

- "密集布局，高信息密度"
- "5-8个核心要点，紧凑排列"
- "使用编号、图标、分隔线组织信息"
- "知识卡片风格，适合学习"
- "信息丰富但不杂乱"
