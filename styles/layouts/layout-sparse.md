# Layout: Sparse (稀疏布局)

## 特点

- **信息密度**：极低（1-2个核心要点）
- **视觉冲击**：最大化
- **适用场景**：封面图、重点强调、情感表达

## 布局规则

### 信息量控制
- 最多 1-2 个核心要点
- 大量留白空间
- 单一视觉焦点

### 视觉层次
- 主要元素占据 60-80% 画面
- 次要元素极少或无
- 强烈的视觉对比

### 文字处理
- 标题：大字号，醒目
- 正文：极少或无
- 关键词：1-3个

## 适用内容类型

- 封面图
- 情感表达
- 品牌宣传
- 重要公告
- 视觉冲击型内容

## 示例结构

```
┌─────────────────────────┐
│                         │
│                         │
│      [核心视觉]          │
│                         │
│      [标题文字]          │
│                         │
│                         │
└─────────────────────────┘
```

## Prompt 提示词

在生成图片时，添加以下约束：

- "极简布局，大量留白"
- "单一视觉焦点，最多1-2个要点"
- "主要元素占据画面中心60-80%"
- "文字极少，仅核心标题"
- "强烈视觉冲击力"
