# Layout: Comparison (对比布局)

## 特点

- **信息密度**：中等（2-4组对比）
- **左右对比**：适合 A vs B、优缺点、前后对比
- **适用场景**：产品对比、方案选择、优缺点分析

## 布局规则

### 信息量控制
- 2个主体对比（A vs B）
- 每个主体 2-4个要点
- 清晰的分隔线或区域

### 视觉层次
- 左右对称布局
- 中间分隔线
- 使用颜色区分两侧

### 文字处理
- 标题：对比主题
- 左侧：A 方案/产品/观点
- 右侧：B 方案/产品/观点
- 每侧 2-4个要点

## 适用内容类型

- 产品对比
- 方案选择
- 优缺点分析
- 前后对比
- A vs B
- 新旧对比

## 示例结构

```
┌─────────────────────────┐
│     [对比标题]           │
├───────────┬─────────────┤
│  [A方案]  │  [B方案]    │
├───────────┼─────────────┤
│ • 要点1   │ • 要点1     │
│ • 要点2   │ • 要点2     │
│ • 要点3   │ • 要点3     │
│ • 要点4   │ • 要点4     │
├───────────┴─────────────┤
│     [结论/建议]          │
└─────────────────────────┘
```

## Prompt 提示词

在生成图片时，添加以下约束：

- "对比布局，左右对称"
- "中间清晰分隔线"
- "使用不同颜色区分两侧"
- "每侧2-4个要点，对应排列"
- "适合 A vs B、优缺点对比"
