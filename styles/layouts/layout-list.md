# Layout: List (列表布局)

## 特点

- **信息密度**：中高（4-7项）
- **枚举排名**：适合清单、排行、步骤
- **适用场景**：Top N、清单、步骤说明

## 布局规则

### 信息量控制
- 4-7 个列表项
- 每项简短说明
- 清晰的序号或标记

### 视觉层次
- 列表项垂直排列
- 每项占据相等空间
- 使用编号、图标、颜色区分

### 文字处理
- 标题：列表主题
- 列表项：简短标题 + 1-2句说明
- 使用序号：1, 2, 3 或 ①②③

## 适用内容类型

- Top N 排行
- 清单列表
- 步骤说明
- 要点枚举
- 检查清单
- 推荐列表

## Prompt 提示词

在生成图片时，添加以下约束：

- "列表布局，垂直排列"
- "4-7个列表项，清晰编号"
- "每项包含标题和简短说明"
- "使用图标或颜色区分各项"
- "适合枚举、排行、步骤"
